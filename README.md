# 📦 3box: Free Source 2 Development

**3box** is the premier, community-driven toolset for developers looking to harness the power of **Valve's Source 2 engine** through the *s&box* editor—completely free of charge.

---

## 🚀 Why Choose 3box?

We believe in democratizing game development. Whether you are a hobbyist or an aspiring developer, 3box provides an accessible gateway to professional-grade tools.

*   **🆓 100% Free:** No subscriptions, no hidden fees. We providde SteamAutoCracker to dissconnect the s&box editor from the Steam ecosystem, so you can use it without paying.
*   **⚖️ Legal & Transparent:** *s&box* is released under the MIT license, which means we can do whatever we want with the code. We have chosen to crack it and modify it slightly for our user's. _This is legal._
*   **⚙️ Professional Engine Access:** Leverage the raw power of **Source 2**—the same industry-leading technology powering titles like *Half-Life: Alyx* and *Counter-Strike 2*—in an intuitive, streamlined package.

---

## 🛠️ Features

*   **Easy Setup:** Our installer automatically clones the _official_ s&box repo and compiles the source code. At the moment, the user is briefly required to use a bundled, third-party application we cannot automate yet, but then the installer handles the rest.
*   **Unmodified Functionality** We don't add or remove things to the s&box editor, so if you decide to buy the offical version, you won't feel a difference. *Note*: We may remove the Publish option from the editor as, without being connected to offical Steam servers, this is virtually useless.
*   **Community-First:** Built by the community, for the community.
